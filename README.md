*Nội dung game
- Bạn điều khiển máy bay tấn công các quái vật xuất hiện trên màn hình.
- Chúng sẽ cố tấn công để tiêu diệt bạn.


*Cách cài đặt:
- Cần mingw32
- Cài đặt thư viện SDL2, SDL2_image, SDL2_ttf, SDL2_Mixer vào Code::Blocks.

- Build & Run

- F hoặc click: Bắn đạn
